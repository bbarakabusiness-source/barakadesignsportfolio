Just Use It Poster Design

Client: Concept Project

Type: Poster Design

Tools: Illustrator, Photoshop

Project Snapshot

This poster design project was created to communicate a clear public health message using a bold, memorable visual style. The design balances humour with clarity so the campaign stands out quickly.

Overview

The project centered on designing an awareness poster for International Condoms Day aimed at promoting safe sex. The banana visual and slogan create immediate recognition and recall.

Outcome

The project resulted in a humorous, attention-grabbing poster that effectively promotes safe sex.
